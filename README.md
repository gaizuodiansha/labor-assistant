# 产程助手 - 宫缩记录工具

一款纯前端、本地存储的宫缩记录工具，帮助产妇在分娩过程中准确记录宫缩规律。

## 功能特性

- ✅ 大按钮快速记录宫缩起止时间
- ✅ 实时宫缩规律图（Canvas 绘制）
- ✅ 数据统计（平均间隔、持续时间、疼痛强度）
- ✅ 宫缩历史列表（可编辑、删除）
- ✅ 疼痛强度评分（1-10级）
- ✅ 伴随事件标记（破水、见红、想用力）
- ✅ 产程管理与历史记录
- ✅ 数据导出/导入（JSON 格式）
- ✅ 智能合并（误操作检测）
- ✅ 震动反馈支持

## 使用方法

1. 直接用浏览器打开 `index.html` 文件
2. 感到疼痛时点击大按钮记录开始
3. 疼痛结束时再次点击记录结束
4. 选择疼痛强度（可选）
5. 随时查看宫缩规律图和统计数据

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- IndexedDB 本地数据存储
- Canvas 图表绘制
- PWA 支持（可安装到主屏幕）

## 兼容性

- iOS Safari 12+
- Android Chrome 70+
- 现代桌面浏览器
